# isonline
A simple Rust program that checks if ip address in `.addresses.txt` are responding to `ping`

## Example Output
```192.168.32.143   is up •
192.168.32.54  is dn •
192.168.32.65  is dn •
192.168.32.27  is up •
192.168.32.43  is up •
192.168.32.233 is up •
```
